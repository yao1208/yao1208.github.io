自我介绍

我是一个2019年9月入学的一名大学生(/ω＼)，你们可以叫我曜，我希望未来在这个博客上记录自己的学习过程。

我喜欢各种有意思的东西。如果你也是这样，我想我们会成为好朋友的。有兴趣的朋友可以给我留言，不过留言需要先注册 GitHub 账号。这是个很有意思的网站，至少它可以搭建一个像我这个样子的blog。

我的计划是在学好我们专业的内容的同时，安排好自己课外的学习。我可能会在这个blog中分享我所学所见，或者一些其他的想法。

说真的，以我现在的水平，搭建这个网页真的耗费了我好多的精力。不过最难的搭建我都完成了，markdown 也不算太难。

未来我会保持更新，希望看到的朋友可以给我留言。感谢！！！(❁´◡`❁)
